Obtain the measurements/data specific to this MovementSensor. If a sensor is not configured to have a measurement or fails to read a piece of data, it will not appear in the readings dictionary. The returns dictionary contains the following readings and values: position: GeoPoint, altitude: double, linear_velocity: Vector3, angular_velocity: Vector3, linear_acceleration: Vector3, compass: double, orientation: Orientation,
